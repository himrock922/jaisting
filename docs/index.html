Jaistingについて

JaistingはFreeBSD OSおける準仮想化機構である「jail」コンテナをブラウザの操作で取り扱うことができるOSSのフレームワークです。

jailコンテナ上で様々な本番環境を構築・デプロイできるようにダッシュボード経由でネットワーク・ファイアウォール・コンソール等の機能を提供しています。 　

　　

ダッシュボードで行える機能は現在も絶賛開発中です。パブリックリリースしたWebサイトもありますので、是非Jaistingによるコンテナ生活をお楽しみ下さい。

各種機能

Jailコンテナの管理

Webブラウザによる操作でJailコンテナの作成・起動・削除ができます。

また、Jailを構築するために必要なFreeBSD OSのソースもブラウザの操作でインストールできます。

ネットワークの設定

FreeBSDホストとJailコンテナをブリッジ接続することにより、Jailコンテナに自由なネットワークを設定することができます。

ファイアウォールの設定

ブラウザ上でFreeBSDホストのファイアウォールの設定を行うことができます。

これにより、Jailネットワークへの通信はNAT/NAPT変換により対外接続をすることができます。

ブラウザターミナルによるJailコンテナの操作

ブラウザ上で実装した疑似ターミナル経由でJailコンテナへUNIXコマンドの実行ができます。
